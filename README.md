# Malla Reddy College Notifications Portal

A React-based campus notification web application built to help students stay updated with important college announcements such as exam results, fee payment notices, placement drives, and faculty instructions.

## Overview

The **Malla Reddy College Notifications Portal** is a simple and responsive web application designed for students to quickly access essential campus updates in one place. The project focuses on improving communication between the college and students by organizing important notifications in a clean and user-friendly interface.

This application can be used to display:
- Exam result announcements
- Fee payment reminders
- Faculty instructions
- Placement notifications
- Event and seminar updates
- Academic and administrative notices

## Features

- Clean and responsive React interface
- Notification cards for quick information access
- Sections for student-related campus updates
- Easy-to-understand layout for important announcements
- Built using Vite for fast development and performance
- Can be extended into a full college notification management system

## Current Notification Categories

The project is designed to support notifications such as:
- **Exam Results** – Semester and internal examination result announcements
- **Fee Payments** – Tuition fee deadlines, payment confirmations, and pending dues alerts
- **Faculty Instructions** – Notices from faculty regarding assignments, attendance, lab work, and class schedules
- **Placement Updates** – Campus recruitment drives, eligibility details, and registration information
- **Events** – Workshops, seminars, fests, and college competitions

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Vardhan576/2311cs040180.git
cd 2311cs040180
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

### 5. Preview production build

```bash
npm run preview
```
